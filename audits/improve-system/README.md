# Weekly System-Improvement Audits

Reports written by the `/improve-system` skill (one per week, dated
`IMPROVE_SYSTEM_<YYYY-MM-DD>.md`). Each is an **INTERNAL** founder document:
eight-lens audit (engineer · auditor · attorney · banker · VC · advisor · board ·
fiduciary) of system health, integrity/non-circularity, agent productivity, and
accuracy vs external ground truth (viral Google Trends, 20% market movers, 20%
crypto movers), with ranked findings + recommendations. Read-only / flag-never-force.
